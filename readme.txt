Init Project
mkdir hello-cli
cd hello-cli
npm init

Run the project
node . -n "name" <? -s searchTerm ?> <? --version ?>

Install as console command
npm install -g .

Uninstall the console command
npm uninstall -g console-template

Run Command
console-template -n "name" <? -s searchTerm ?>
